# 弹幕小游戏
#

工程文件是Visual Studio 2013的工程文件，使用C#编写，可执行文件在Debug和Release中都有，要使用的话，使用Release版本

食用方法：←或A控制方块左移， →或D控制方块右移， 空格键发射子弹，ESC退出游戏
